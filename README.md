# FTL
Ce projet fournit un système d’overlay interactif pour le jeu Faster Than Light (FTL), permettant de gérer facilement les sauvegardes et restaurations de parties en cours. L’overlay s’affiche par-dessus la fenêtre du jeu et offre une interface compacte, transparente et toujours visible, contrôlable au clavier (pavé numérique et flèches).
